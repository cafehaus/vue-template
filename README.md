# vue 前端 PC 网站开发模板

### main 分支
vue3 + vite + TypeScript + vue-router + pinia + eslint

### vue2-iview 分支

网站模板管理端风格：vue2 + iView + vue-router + vuex + axios + webpack5 + stylus

#### 安装依赖
```
npm install
```

#### 启动开发服务
```
npm run dev
```

#### 构建代码
```
npm run build
```

#### 修正代码风格
```
npm run lint
```
